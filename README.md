# 💸 Smart Expense Tracker API

A scalable backend REST API for managing personal expenses, built using **Node.js, Express.js, and MongoDB**.  
Includes secure authentication, modular architecture, and efficient async processing.

---

## 🚀 Features

- 🔐 JWT-based Authentication (Login/Register)
- 📊 Expense Management (CRUD APIs)
- ⚡ Asynchronous request handling with async/await
- 🧱 Clean MVC Architecture (Controllers, Models, Routes)
- 🔒 Password hashing using bcrypt
- 🌐 RESTful API design
- 📦 Scalable and production-ready structure

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Authentication:** JWT (jsonwebtoken), bcrypt  
- **Tools:** Nodemon, dotenv
